# githubFullWidthNewFiles (Deprecated)

[Install](https://github.com/ad08fee3/userscripts/raw/refs/heads/main/userscripts/deprecated/githubFullWidthNewFiles/githubFullWidthNewFiles.user.js)

> Deprecated because Refined GitHub now covers this behavior. At the time of writing, this script still works, but I will no longer be personally using/updating it.

On PR, commit, and compare diffs, makes wholly-added or wholly-deleted files use the full portal width in side-by-side view by collapsing the empty side's columns. A file with no deletions (or no additions) no longer wastes half the screen on a blank column. It handles both the new React diff UI and the classic/legacy diff UI served on commit, compare, and Enterprise pages.
