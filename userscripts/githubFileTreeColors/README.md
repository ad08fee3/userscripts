# githubFileTreeColors

[Install](https://github.com/ad08fee3/userscripts/raw/refs/heads/main/userscripts/githubFileTreeColors/githubFileTreeColors.user.js)

Colors the PR sidebar file tree to reflect each file's review state at a glance.

- **Collapsed** files are shown in gray italics.
- **Viewed** files get a blue strikethrough.
- **Viewed and collapsed** files get both: gray italics with a blue strikethrough.

Folder names in the tree follow the same rules, derived from the aggregate state of all files they contain.

Works in both light and dark mode.
