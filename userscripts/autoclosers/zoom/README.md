# autoCloseZoom

[Install](https://github.com/ad08fee3/userscripts/raw/refs/heads/main/userscripts/autoCloseZoom/autoCloseZoom.user.js)

Automatically closes the Zoom meeting launch tab once the app launches.

When you click a Zoom meeting link, your browser opens a launch page before handing off to the Zoom app. This script overlays a countdown and closes the tab once it reaches zero.

This was inspired by https://github.com/ChromeAdmin/meetings-page-auto-closer-for-zoom
