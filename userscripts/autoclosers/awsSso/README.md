# autoCloseAwsSso

[Install](https://github.com/ad08fee3/userscripts/raw/refs/heads/main/userscripts/autoCloseAwsSso/autoCloseAwsSso.user.js)

Automatically closes the AWS SSO tab after credentials have been shared.

After logging in with AWS SSO, you get redirected to a page at `127.0.0.1`, which confirms that you have logged in. This script overlays a countdown and closes the tab once it reaches zero.
