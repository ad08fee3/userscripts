# githubCollapsibleHeaderBars

[Install](https://github.com/ad08fee3/userscripts/raw/refs/heads/main/userscripts/githubCollapsibleHeaderBars/githubCollapsibleHeaderBars.user.js)

Makes GitHub's header bars fully clickable to collapse their content (diff file headers, inline comment threads, and PR discussion comment threads) instead of requiring you to click the small chevron button. This applies across PR files, commit pages, compare pages, and PR discussion threads.

The browser extension _Refined GitHub_ provides similar functionality on a subset of pages. This userscript is designed to work in parallel and can work with or without that extension installed.
